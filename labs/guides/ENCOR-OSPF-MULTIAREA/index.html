Overview

This lab focuses on the ENCOR 3.2.b multi-area requirements: multiple normal areas, ABR behavior, inter-area summarization, and filtering. You will configure areas 0, 10, and 20, then reduce route table size without hiding required application prefixes.

Task 2: Configure Multiple Normal Areas

Place transit links and LANs in the correct areas. Do not use stub or NSSA area types; the exam topic calls out multiple normal areas.

router ospf 10
 router-id 1.1.1.1
 passive-interface default
 no passive-interface ethernet0/0
 no passive-interface ethernet0/1
 network 10.0.0.0 0.0.0.255 area 0
 network 10.10.0.0 0.0.255.255 area 10
show ip ospf neighbor
show ip ospf interface brief
show ip ospf database summary
show ip route ospf

Task 3: Summarize at ABRs

Summarize only on ABRs and only in the direction where the summary should be advertised. Use area range for inter-area summaries.

router ospf 10
 area 10 range 10.10.0.0 255.255.0.0
 area 20 range 10.20.0.0 255.255.0.0
!
show ip ospf database summary 10.10.0.0
show ip route 10.10.0.0 255.255.0.0

Depth check: area range summarizes routes between OSPF areas. summary-address is for external routes redistributed into OSPF.

Task 4: Filter Inter-Area Routes

Filter a lab-only service prefix from leaving area 20 while preserving the required branch summary. Use a prefix list and area filter-list.

ip prefix-list BLOCK-LAB-ONLY seq 5 deny 10.20.99.99/32
ip prefix-list BLOCK-LAB-ONLY seq 100 permit 0.0.0.0/0 le 32
!
router ospf 10
 area 20 filter-list prefix BLOCK-LAB-ONLY out
!
show ip ospf database summary
show ip route 10.20.99.99
show ip protocols | section Routing for Networks

Filtering inter-area routes affects Type-3 LSAs. It is not the same as distribute-list filtering of routes from the local RIB.

Task 5: Validation and Troubleshooting

  1. R5-SVC is accidentally placed in area 0 on one side and area 10 on the other; identify the database symptom.
  2. R1-HQ summarizes area 10 with a /15 and hides a branch aggregate; correct the range.
  3. A prefix list denies all Type-3 LSAs because the final permit is missing.
